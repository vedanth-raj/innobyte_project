## üé≠ Animation Timeline

**On Page Load:**
1. Hero content fades in (0.5s)
2. Navigation appears (0.3s)
3. Scroll indicator animates

**On Scroll:**
1. Navigation becomes solid
2. Sections fade in as they enter viewport
3. Counters animate when visible
4. Cards appear with stagger effect

**On Hover:**
1. Images zoom (0.5s)
2. Cards lift (0.3s)
3. Buttons glow
4. Icons rotate
5. Shadows intensify

**On Click:**
1. Ripple effect from click point
2. Smooth scroll to section
3. Form validation feedback
4. Lightbox opens with fade
