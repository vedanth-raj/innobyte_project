## Migration Guide

### From v1.0.0 to v2.0.0

1. **Backup old files**
   ```bash
   cp index.html index.html.backup
   cp css/styles.css css/styles.css.backup
   cp main.js main.js.backup
   ```

2. **Replace files**
   - Replace `index.html` with new version
   - Replace `css/styles.css` with new version
   - Replace `main.js` with new version

3. **Update content**
   - Review and update text content
   - Verify contact information
   - Check all links
   - Update images if needed

4. **Test thoroughly**
   - Test on multiple devices
   - Check all interactions
   - Verify forms work
   - Test all links

5. **Deploy**
   - Follow DEPLOYMENT.md guide
   - Test live site
   - Monitor performance
